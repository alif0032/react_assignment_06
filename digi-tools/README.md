# About Projects
 # DigiTools

## Project Name
DigiTools - Digital Tools Platform

## Description
This is a digital tools website where users can explore different tools and add them to cart. Users can switch between product view and cart view easily. It is designed based on a modern UI and focuses on a simple user experience.

## Technologies Used
- React.js  
- Tailwind CSS  
- DaisyUI  
- JavaScript (ES6)  
- React Toastify  

## Features
1. Users can add products to the cart and remove them anytime  
2. Product and Cart toggle system (switch view easily)  
3. Button changes and shows alert when product is added
